---

## 🔑 API Used

This app uses the free tier of **WeatherAPI**:

```
http://api.weatherapi.com/v1/current.json?key=YOUR_API_KEY&q=London&aqi=yes
```

Replace `YOUR_API_KEY` with your personal key from [weatherapi.com](https://www.weatherapi.com/).
